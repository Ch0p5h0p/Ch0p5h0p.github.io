# Portfolio

## [Ch0p Origins](https://github.com/Ch0p5h0p/Ch0pOrigins)
My custom-built origins plugin for a Paper Minecraft server. Still a work-in-progress. It includes custom origins suggested and planned out by my friends

## [MCTools](https://github.com/Ch0p5h0p/MCTools)
A Fabric mod that implements varying tech-oriented tools into Minecraft. Also still a work-in-progress. So far, it includes:
- Grep
- AWK scripting, with an API to run in-game functions
- Script editor

Planned features include:
- GZip compression
- SHA-256 hashing
- AES-256-GCM encryption
- Maybe another scripting language aside from AWK

## [MCJS](https://github.com/Ch0p5h0p/MCJS)
A Fabric mod (noticing a trend?) implementing JavaScript programming into Minecraft, with help from the QuickJS library. Includes an API to interact with the game and a script editor

## [CryptoCraft](https://github.com/Ch0p5h0p/CryptoCraft)
Yet another Fabric mod implementing cryptography into minecraft. Includes:
- AES-256-CBC
- RSA-1024 encryption
- SHA-256 hashing

## [ChatAutoMod](https://github.com/Ch0p5h0p/ChatAutoMod)
A plugin for a paper server to automate response to restricted words, including censoring and punishing. Includes:
- Configurable levels of severity
- Configurable bad word list
- Handling for (most) forms of self-censoring

## [SuperSickStats](https://github.com/Ch0p5h0p/SuperSickStats)
A fabric mod giving detailed information about the armor you're wearing, including overall percent damage reduction and percent reductions for each type of protection. Also includes calculators for DPS and for spear damage at different speeds.
